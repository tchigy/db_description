Database Description

add description to your database in your native language

Installation

Adding the module to your addons-path it's not enough in order to give an application name to your first database. you need to add this configuration in your config file.

server_wide_modules = web,web_kanban,db_description

This way you tell odoo to load the module when there is no database in the system.

Create a new database

You will notice a new field Application Name in this field you can write any thing you want in any language.

data base selecting

Instead of showing the IT name of the database Odoo will show the Application name.

Rename Application

You can change the description by clicking on rename app link

Login page

In your login page if the module is installed in the selected database. now it shows witch application you selected instead of the database name.
